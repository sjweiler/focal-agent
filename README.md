Focal-Agent aims to be a practical AI desktop assistant capable of:

- Running completely offline using local LLMs
- Connecting to cloud LLM providers when desired
- Executing user-approved system actions
- Supporting third-party plugins
- Understanding voice commands
- Remembering context between conversations
- Integrating naturally into Windows
